# 贡献者行为准则

## 我们的承诺

为了营造一个开放和欢迎的环境，我们作为贡献者和维护者承诺：参与我们项目和社区的每个人都不会因为年龄、身体特征、残疾、种族、性别认同和表达、经验水平、国籍、外表、种族、宗教或性别认同和性倾向而受到骚扰。

## 我们的标准

有助于创建积极环境的行为包括：

- 使用欢迎和包容的语言
- 尊重不同的观点和经验
- 接受建设性的批评
- 专注于对社区最有利的事情
- 对其他社区成员表示同情

不可接受的行为包括：

- 使用性化的语言或意象以及不欢迎的性关注或进步
- 挑衅、侮辱性或贬低的评论，以及人身或政治攻击
- 公开或私下骚扰
- 在未明确同意的情况下发布他人的私人信息，例如物理地址或电子地址
- 在专业设置中被合理认为不恰当的其他行为

## 执行

项目维护者负责阐明可接受的行为标准，并应对任何不可接受行为采取适当的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、问题和其他贡献，或者在违反此行为准则时暂时或永久禁止任何贡献者。

## 范围

本行为准则适用于：

- 项目空间内的所有行为
- 代表项目或社区的公共空间

## 举报

如果你目睹了不可接受的行为，请将详细信息报告给项目负责人。所有投诉将被审查和调查。

## 执行准则

项目维护者将遵循这些社区影响准则来确定他们对任何被认为违反了这一行为准则的行为的后果：

### 1. 纠正

**社区影响**：使用不恰当的语言或其他被认为不专业或不受欢迎的行为。

**后果**：来自社区领导者的私下书面警告，澄清违规的性质和为什么该行为不恰当。可能会要求公开道歉。

### 2. 警告

**社区影响**：通过单一事件或一系列行为的违规。

**后果**：带有后果警告的书面警告。对相关人员的进一步互动可能受到限制，例如临时禁止与行为准则强制者进行互动。

### 3. 临时禁止

**社区影响**：严重违反社区标准，包括持续的不适当行为。

**后果**：临时禁止与社区的任何类型的互动或通信。在此期间禁止与相关人员的任何公开或私下互动，包括与强制执行者的互动。违反这些条款可能导致永久禁止。

### 4. 永久禁止

**社区影响**：表现出对社区标准的持续违规模式，包括持续不当行为、骚扰个人或对个人群体的侵犯。

**后果**：永久禁止参与社区的任何形式的公开互动。
